# `.clawcode/learning/instincts/`

**Instinct** markdown files: compact, learned rules distilled from observations (`observations.jsonl`) or imports.

## Subfolders

| Path | Purpose |
|------|---------|
| [`personal/`](./personal/) | **Locally learned** instincts (e.g. `learned-<timestamp>.md`). |
| [`inherited/`](./inherited/) | **Imported** instincts from teammates or external bundles (e.g. `import-<timestamp>.md`). |

See `learning/paths.py`, `learning/service.py`, and slash commands around instinct list/export/import.
