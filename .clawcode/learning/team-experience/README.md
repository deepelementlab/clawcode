# `.clawcode/learning/team-experience/`

**Team-scoped** experience: handoff capsules, exports, and feedback logs for multi-developer workflows (`/team-experience-*` slash commands).

## Subfolders & files

| Path | Purpose |
|------|---------|
| [`capsules/`](./capsules/) | Serialized **capsules** for share/apply flows. |
| [`exports/`](./exports/) | **Exports** produced for import elsewhere. |
| `feedback.jsonl` *(when present)* | Append-only **feedback** log for team experience tuning. |
| `deeploop_iterations.jsonl` *(when present)* | Deep-loop iteration log for team runs. |

See `LearningPaths` and `learning/service.py`.
