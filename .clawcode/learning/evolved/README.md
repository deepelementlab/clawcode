# `.clawcode/learning/evolved/`

**Artifacts produced by evolution / promotion** from the closed-loop learning pipeline: skills, slash commands, and subagent definitions promoted from experience.

## Subfolders

| Path | Purpose |
|------|---------|
| [`skills/`](./skills/) | Evolved **skill** trees (`SKILL.md` per skill folder). |
| [`commands/`](./commands/) | Evolved **custom command** markdown (`.md`). |
| [`agents/`](./agents/) | Evolved **subagent** role markdown (`.md`). |

See `LearningPaths` in `learning/paths.py` and promotion logic in `learning/service.py`.
