# `.clawcode/learning/experience/`

**Solo / workspace experience** artifacts parallel to `team-experience/`: capsules, exports, and logs such as `feedback.jsonl` or `instinct_delta.jsonl` under the learning root.

## Subfolders

| Path | Purpose |
|------|---------|
| [`capsules/`](./capsules/) | Experience **capsules** for local handoff and replay. |
| [`exports/`](./exports/) | **Exports** of experience bundles. |

See `learning/service.py` for paths like `experience/feedback.jsonl` and capsule directories.
