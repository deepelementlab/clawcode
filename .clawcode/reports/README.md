# `.clawcode/reports/`

Generated **reports** (JSON and/or Markdown) from observability and backend flows — e.g. ops summaries emitted under `Path.cwd() / ".clawcode" / "reports"` (`ops_observability.py`).

## Usage

- Inspect locally after runs that emit structured reports.
- Filenames often include timestamps or backend identifiers.

## Git

Usually **local/CI-only**; commit only when you want to preserve a specific report in the repo.
